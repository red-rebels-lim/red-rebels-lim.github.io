# Red Rebels Calendar

A sports calendar for Nea Salamina, covering football and volleyball fixtures throughout the season.

## Features

- **Monthly calendar view** -- Browse fixtures by month with easy navigation and a "jump to today" shortcut
- **Match details** -- View opponent, venue, kick-off time, and result for each fixture
- **Statistics dashboard** -- Overall record, home/away splits, recent form, streaks, head-to-head records, goal distribution charts, and season progress
- **Multi-sport support** -- Men's football, men's volleyball, and women's volleyball
- **Filters** -- Filter events by sport, home/away, upcoming/completed, or search by opponent name
- **Export & print** -- Export fixtures to iCalendar (.ics) or print the calendar
- **Countdown timer** -- See how long until the next match
- **Bilingual** -- Available in English and Greek
- **Dark / Light theme**
- **Mobile friendly** -- Responsive layout with swipe navigation
